# Node Performance Test
This repository is only a stupid game to test the performance of node on differents activities.

## Performance Test's
* Collections Test's
  * Immutable JS Tests
    - List push
    - List mapping
  * Array Tests
    - Array push
    - Array iteration
      - for
      - for let x in array
      - forEach
    - Array mapping

## Set up
`npm install`

## Running test's
`npm run test`

## Tech's
* TypeScript
* Immutable
* npm
* `console.time`
